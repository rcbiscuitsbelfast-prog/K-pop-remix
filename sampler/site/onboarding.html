Welcome to the MIFF contributor guide

This zone is fully remix-safe and ready for forks.

Setup Instructions

  1. Clone or fork the repo.
  2. Run npm install and npm test.
  3. Use npm run test:golden and npm run test:remix to validate.

Asset Audit Summary

All assets must be CC0, GPL, or public domain. See assets/README.md for sources.

Remix Hooks & Fixtures

Explore fixtures in sampler/scenarios/ and zone code in sampler/zones/. Use Pure modules for deterministic behavior.

Tip: Enable Remix Mode on the landing page to test overlays.
